# 已废弃：旧角色 Prompt

当前项目不再使用本文件驱动 Agent 决策。

新的 Agent 决策统一在 `src/agent_tools.py` 中完成：
- 引擎按当前阶段只暴露一个工具集合。
- Agent 每一轮只能返回一个 `tool_call`。
- 发言也必须通过 `speak_public` 或 `werewolf_private_message` 工具完成。
- 白天公开发言不能泄漏狼人私聊、队友、刀口、未公开查验、未公开用药。
- 狼人公开发言需要伪装成好人视角；狼人真实策略只允许出现在狼人私聊工具中。

请不要重新把本文件接回主流程；如需调整角色行为，请修改 `ROLE_GUIDANCE`、`INTENT_INSTRUCTIONS` 和工具定义。
